# LoadLink – Backend-Driven Truck Booking Platform

LoadLink is a backend-focused logistics platform that allows users to book trucks and drivers to manage delivery requests through a secure workflow system.

The project was built to demonstrate backend engineering skills such as API development, authentication, database design, middleware, and deployment.

## Key Features

* User Registration & Login
* Driver Registration & Login
* JWT Authentication
* Role-Based Access Control
* Create Booking Requests
* Driver Accept Booking Flow
* Delivery Status Workflow

```text
Pending → Accepted → In Transit → Completed
```

* MongoDB Database Integration
* Protected Routes & Middleware
* Fully Deployed Project

## Run Locally

```bash
git clone https://github.com/Pawan116/LoadLink.git
cd LoadLink/backend
npm install
npm run dev
```

Create `.env`

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
```
